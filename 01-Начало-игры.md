# 🚀 Начало игры

[← На главную](README.md)

## 1. Как зайти

- **Версия:** Minecraft **1.21.11**
- **Клиент:** любой (ванильный подойдёт). Моды ставить не нужно — весь кастомный контент приходит через ресурспак сервера.
- **Ресурспак:** согласись загрузить, когда сервер предложит. Без него не будут видны кастомные текстуры (мебель, иконки, шрифты рангов).

> Сервер по **вайтлисту**. Если тебя выкидывает с сообщением про whitelist — напиши администрации, чтобы добавили твой ник.

## 2. Регистрация и вход

Твой аккаунт защищён паролем. При первом заходе — зарегистрируйся, дальше просто входи:

```
/register <пароль> <пароль>   ← первый раз
/login <пароль>               ← каждый следующий вход
```

Сменить пароль: `/changepassword <старый> <новый>`

⚠️ Никому не сообщай свой пароль — администрация его никогда не спрашивает.

## 3. Первые шаги

1. **Авторизуйся** (`/login`).
2. Осмотрись на спавне: телепорт `/warp spawn` или `/spawn`.
3. Поставь **дом**, чтобы возвращаться: `/sethome` (первый дом), позже `/home`.
4. Найди место для базы и осваивайся.
5. Открой для себя фишки сервера — начни с [Навыков](04-Навыки.md), [Питомцев](03-Питомцы.md) и [Экономики](05-Экономика.md).

## 4. Что делает игру на AoiWorld особенной

- 🦊 **Лисы Аой** — фирменные питомцы сервера ([подробнее](03-Питомцы.md))
- ⭐ **RPG-навыки** с прокачкой, коллекциями и статами
- 💰 **Экономика с казино** — заработок, магазины, блэкджек
- 🌾 Огромный набор еды, ферм, мебели и структур
- 🎙️ **Голосовой чат** и эмоции для настоящей движухи

## 5. Совет

Не помнишь команду? Введи `/` и жми **Tab** — клиент покажет доступные команды и подсказки к ним. Это работает для всего на сервере.
