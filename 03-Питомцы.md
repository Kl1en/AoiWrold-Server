# 🦊 Питомцы

[← На главную](README.md)

На AoiWorld целых **три** системы питомцев. Главная — фирменные **Лисы Аой**.

## 🌸 Лисы Аой (AoiPets)

Кастомная система, написанная специально для сервера (в духе MCPets). Лиса ходит за тобой, как прирученный волк, с моделькой и статусом над головой («Следует за хозяином» / «Сидит» / «Атакует»).

**6 лис:**
| ID | Лиса |
|----|------|
| `red_fox` | Красная |
| `fall_fox` | Осенняя |
| `gray_fox` | Серая |
| `pink_fox` | Розовая |
| `purple_fox` | Фиолетовая |
| `arctic_fox` | Арктическая |

**Команды:**
```
/aoifox                    — открыть меню выбора лисы
/aoifox summon <лиса>      — призвать
/aoifox dismiss            — отозвать
/aoipet                    — то же самое (алиас)
```

**Доступ по рангам.** Лисы выдаются через права (строгий режим — «всё по рангам»):
- `aoipets.use` — базовый доступ к меню
- `aoipets.summon.<лиса>` — конкретная лиса (напр. `aoipets.summon.arctic_fox`)
- `aoipets.summon.*` — все лисы сразу

В меню лисы, к которым у тебя нет доступа, показаны как заблокированные (🚫 «Нет доступа»). Хочешь редкую лису — смотри, как её получить (ранг/донат/награда) в анонсах сервера.

## ⭐ Петы навыков (PeakSkills)

Система навыков [PeakSkills](04-Навыки.md) имеет **собственных питомцев**, привязанных к прокачке. Их можно активировать/деактивировать через меню навыков (`pets`). Они дают бонусы по мере роста уровней.

## ⚔️ Боевые лисы (Battle Fox)

Отдельный мод — **сражающиеся лисы-компаньоны с RPG-прокачкой**. В отличие от декоративных Лис Аой, боевые лисы участвуют в бою и растут в силе. Подробнее о боевом применении — в разделе [PvP и бои](14-PvP.md).

---

**Итого:** Лисы Аой — для красоты и компании 🦊, петы навыков — для бонусов ⭐, боевые лисы — для драк ⚔️.
