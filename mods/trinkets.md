# 💍 Trinkets — аксессуары

[← Каталог модов](README.md)

Система **слотов для аксессуаров** — надевай кольца, амулеты, пояса и прочее в отдельные ячейки, не занимая руки и броню. Открой экран аксессуаров (кнопка рядом с инвентарём).

## 🎒 Слоты

| Слот | Куда |
|------|------|
| **Face** | лицо (маски/очки) |
| **Hat** | головной убор |
| **Necklace** | ожерелье/амулет |
| **Back** | спина |
| **Cape** | плащ |
| **Belt** | пояс |
| **Aglet** | шнурки/акцент на обуви |
| **Shoes** | обувь |
| **Glove / Offhand Glove** | перчатки (осн./вторая рука) |
| **Ring / Offhand Ring** | кольца (осн./вторая рука) |

Некоторые слоты имеют **несколько ячеек** (напр. пара колец).

## 🔧 Как работает

- Trinkets — это **основа (framework)**: слоты предоставляет он, а сами аксессуары добавляют другие моды сервера.
- Предмет с пометкой «Equippable in the trinket slots» надевается в соответствующий слот и даёт свой эффект **пассивно**, пока надет.
- Аксессуары не мешают броне и оружию — это отдельные ячейки.

## 💡 Советы

1. Открой экран аксессуаров и посмотри доступные слоты.
2. Ищи предметы с подписью про trinket-слот — это аксессуары.
3. Комбинируй эффекты аксессуаров под свой билд (в связке с [навыками](peakskills.md) и [зачарованиями](neoenchant.md)).
