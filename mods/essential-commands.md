# 🏠 Essential Commands — дом, варпы, телепорты

[← Каталог модов](README.md) · [Обзор перемещения](../06-Дом-и-варпы.md)

Базовые команды для комфортной игры: дома, телепорты между игроками, варпы, спавн, возврат и случайная телепортация.

## 🏡 Дома
| Команда | Действие |
|---------|----------|
| `/sethome [имя]` | поставить дом (первый — без имени) |
| `/home [имя]` | телепорт домой |
| `/homes` | список твоих домов |
| `/delhome <имя>` | удалить дом |

**Лимит домов зависит от ранга.** Дополнительные дома ставятся с именем: `/sethome ферма`, `/sethome шахта`.

## 🌐 Варпы и спавн
| Команда | Действие |
|---------|----------|
| `/spawn` | телепорт на спавн |
| `/warp <имя>` | публичный варп |
| `/warps` | список варпов |

## 🤝 Телепорт к игрокам
| Команда | Действие |
|---------|----------|
| `/tpa <игрок>` | запросить тп к игроку |
| `/tpahere <игрок>` | позвать игрока к себе |
| `/tpaccept` · `/tpdeny` | принять / отклонить запрос |

Запросы работают по подтверждению — тебя не телепортируют без согласия.

## 🔙 Прочее
| Команда | Действие |
|---------|----------|
| `/back` | вернуться на прошлую точку (после смерти/тп) |
| `/rtp` | случайная телепортация в дикий мир |
| `/nick <ник>` | сменить отображаемый ник |

## 💡 Механика и советы

- **Кулдауны и задержки.** У телепортов может быть кулдаун и «прогрев» (нельзя двигаться пару секунд перед тп) — это защита от абуза в бою.
- `/back` — твой друг: вернёт к месту гибели, чтобы забрать вещи из [могилы](universal-graves.md).
- Комбинируй с [вейстоунами](waystones.md): дома — для личных точек, вейстоуны — для публичной сети.
- `/rtp` удобен, чтобы начать базу вдали от спавна в свежих чанках.
