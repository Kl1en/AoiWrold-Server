# 🗄️ Ящики и хранилища

[← Каталог модов](README.md) · [Обзор строительства](../09-Строительство.md)

Несколько систем хранения под разные задачи — от массовых ящиков до тематических шкафов и улучшенных печей.

## 🗃️ Extended Drawers — ящики-накопители

Ящики, каждый из которых хранит **один тип предмета, но в огромном количестве**, с отображением числа на лицевой стороне.
- **Апгрейды вместимости** — увеличивают, сколько предметов влезает в ящик.
- **Компактные ящики** — автоматически конвертируют слиток ↔ блок (9↔1).
- **Контроллер** — доступ ко всей стене ящиков из одной точки/для автоматизации.
- Быстрое **закидывание/выемка** кликами.

Идеально для массового хранения ресурсов (булыжник, дерево, руды) без стены сундуков.

## 🍱 Storage Delight — тематическое хранение

Стильные шкафы и ёмкости (аддон в духе Farmer's Delight):
- **Drawers / Small Drawers** по всем породам дерева — для еды и ресурсов.
- Хорошо сочетается с кухней [Farmer's Delight](farmers-delight.md).

## 🔥 More Furnaces — улучшенные печи

Печи, которые плавят **быстрее** ванильной:
| Печь | Скорость |
|------|----------|
| **Iron Furnace** | 2× |
| **Golden Furnace** | 3× |
| **Diamond Furnace** | 4× |
| **Netherite Furnace** | 6× |

Плюс **модули-модификаторы**:
- **Speed Modifier** (2×–5×) — ещё быстрее, но больше расход топлива.
- **Efficiency Modifier** — экономит топливо ценой скорости.

Настраивай печь под задачу: скорость vs экономия.

## 📦 Другие варианты хранения
- **[Digital Chest](digital-chest.md)** — безлимитное цифровое хранилище с поиском.
- **[Server Backpacks](server-backpacks.md)** — переносные рюкзаки.

## 💡 Советы

1. **Extended Drawers** — для «одного типа много» (руды, блоки).
2. **Digital Chest** — для «всего понемногу» с поиском.
3. **Netherite Furnace + Speed** — для массовой плавки; **Efficiency** — когда мало топлива.
