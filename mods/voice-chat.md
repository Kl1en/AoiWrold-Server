# 🔊 Voice Chat — голосовой чат

[← Каталог модов](README.md) · [Обзор социалки](../12-Социалка.md)

Настоящий **пространственный голосовой чат** + расширения: рации, голосовые сообщения и проигрывание аудио.

## 🎙️ Simple Voice Chat (основа)

- **Пространственный звук:** чем ближе игрок — тем громче; далёких не слышно.
- **Групповой чат:** можно создавать голосовые группы.
- **Настройки:** громкость, микрофон, режим (push-to-talk / голосовая активация).

> Нужен **клиентский мод Simple Voice Chat** (ставится в один клик под Fabric). Без него ты не сможешь говорить/слышать голос.

## 📻 Расширения

| Мод | Что добавляет |
|-----|---------------|
| **Simple Voice Chat Radio** | рации/радио — связь на расстоянии, вне зоны обычного голоса |
| **Voice Messages** | записывай **голосовые сообщения** прямо в чат |
| **Voice Chat Interaction** | голосовое взаимодействие с игровым миром |
| **AudioPlayer** | проигрывание кастомного аудио (диски/устройства) |

## 💡 Советы

1. Поставь **клиентский Simple Voice Chat** — это основа всей голосовой движухи.
2. Настрой **push-to-talk**, чтобы не фонить в общий канал.
3. **Рация** удобна для координации на больших расстояниях (ивенты, стройка).
4. Голосовой чат сильно оживляет совместную игру и стримы — идеально для комьюнити Aoi.
