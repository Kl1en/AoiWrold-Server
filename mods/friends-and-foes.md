# 🐾 Friends&Foes — новые мобы

[← Каталог модов](README.md) · [Обзор мира](../11-Мир.md)

Добавляет мобов из «mob vote» и забытых концептов — новые дружелюбные и враждебные существа с уникальным поведением.

## 🐾 Мобы

| Моб | Кто это |
|-----|---------|
| **Copper Golem** (Медный голем) | нажимает кнопки, со временем окисляется |
| **Glare** (Глар) | показывает тёмные (опасные для спавна) места |
| **Iceologer** (Ледокол) | иллагер, кидает ледяные глыбы |
| **Illusioner** (Иллюзионист) | иллагер-маг с иллюзиями и Player Illusion |
| **Rascal** (Раскал) | играет в прятки, награждает за находку |
| **Moobloom** (Мублум) | цветочная корова |
| **Crab** (Краб) | пляжный моб |
| **Tuff Golem** (Туфовый голем) | держит и охраняет предметы |
| **Wildfire** (Вайлдфайр) | огненный мини-босс Незера |
| **Mauler** | враждебное существо |
| **Beekeeper** | связан с пчёлами |

## 💡 Чем полезны

- **Glare** — подсказывает, где враги могут заспавниться (света мало).
- **Copper Golem** — автоматизация кнопок/редстоуна.
- **Tuff Golem** — «витрина»/охрана предмета в декоре.
- **Rascal** — мини-игра в прятки ради награды.
- **Wildfire** — вызов и лут в Незере.

## 💡 Советы

1. Изучай поведение — многие мобы **полезны**, а не просто декор.
2. Медный голем и туфовый голем интересны для баз и ферм.
3. Осторожно с **Iceologer/Wildfire** — это боевые угрозы.
