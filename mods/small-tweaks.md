# 🧩 Мелкие твики

[← Каталог модов](README.md)

Небольшие моды, каждый из которых чинит одно ванильное неудобство. Работают автоматически или по простому действию.

## 💰 Disable Too Expensive
Убирает надпись **«Слишком дорого!»** в наковальне. Теперь можно комбинировать предметы и зачарования без ванильного лимита стоимости.

## 🌀 NetherPortalFix
Гарантирует, что портал в **Незер и обратно ведёт в правильный портал**, а не в чужой. Критично для мультиплеера — больше не «выкидывает» к соседям.

## 🖼️ Craftable Invisible Item Frames
Позволяет скрафтить **невидимые рамки** (обычная рамка в верстаке). Идеально для чистой выкладки предметов и панно из [Image2Map](image2map.md) без видимых рамок.

## 📋 NBT Copy
Добавляет **кнопку копирования** вывода команды `/data get` — удобно для строителей, картостроителей и техно-игроков, работающих с NBT.

## 🎁 Personal Loot
Персональный лут в сундуках данжей — вынесен на [отдельную страницу](personal-loot.md).
