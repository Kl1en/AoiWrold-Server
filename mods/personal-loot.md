# 🎁 Personal Loot — персональный лут

[← Каталог модов](README.md)

Решает вечную проблему совместных данжей: **лут из сундука персональный** — каждому игроку свой набор. Больше никто не «выгребает всё первым».

## 🔧 Как работает

- В сундуках с лутом (в структурах/данжах) **каждый игрок получает собственную выкатку** предметов.
- Ты открываешь сундук — тебе генерируется **твой** лут, независимо от других.
- Один сундук — награда для всей группы, без конкуренции и обид.

## 📋 Правила взаимодействия

- **Один раз на игрока:** «You already looted this chest» — повторно тот же сундук не фармится.
- **Занятость:** если кто-то уже открыл — «Someone is currently using this chest», подожди.
- **Убрать сундук:** ломается **только в присесте** (Sneak + ломать). Без присеста сломать нельзя — защита от случайной потери.
- **Медные големы** не могут открывать персональные лут-сундуки.

## 💡 Почему это круто

- Ходите в данжи **компанией** — каждому достанется награда.
- Никакого «кто первый добежал», меньше конфликтов.
- Отлично для дружного комьюнити и совместных вылазок за [структурами](structures.md).
