# 🧱 Blockus — строительные блоки

[← Каталог модов](README.md) · [Обзор строительства](../09-Строительство.md)

Больше **1500 декоративных блоков** для строителей. Огромный выбор фактур, кирпичей, плитки и деревянного декора под любой стиль.

## 🧱 Что внутри

- **Кирпичи и камень:** аметистовые, андезитовые и десятки других видов кирпича + плиты/ступени/стены к ним.
- **Деревянный декор:** timber frame (фахверк), решётки (grate/lattice), изгороди (hedge), мозаика, столбы (post) — по всем породам дерева.
- **Мостовые и паттерны:** circular paving и прочие узорные блоки.
- **Освещение:** лампы и фонари (в т.ч. аметистовые).
- **Ящики (Crate)** для ресурсов как декор-хранение.

Практически для каждого материала есть полный набор: **блок → плита → ступени → стена**.

## 💡 Советы

1. Blockus — про **палитру**: используй его фактуры, чтобы уйти от «ванильной» однообразности.
2. Timber frame + мозаика отлично смотрятся на фасадах и в интерьере.
3. Комбинируй с [мебелью](decorations-furniture.md) для завершённого вида.
4. Все виды блоков и крафты — в игре через [Polydex](../13-Качество-жизни.md).
