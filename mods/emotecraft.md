# 💃 Emotecraft — эмоции

[← Каталог модов](README.md) · [Обзор социалки](../12-Социалка.md)

Проигрывай **эмоции** персонажем — танцы, жесты, приветствия. Отлично для комьюнити, ролёвки и стримов.

## 🔧 Как работает

1. Открой **колесо быстрого выбора эмоций** — по умолчанию клавиша **B**.
2. Настрой любимые эмоции в это колесо.
3. Проиграй эмоцию — персонаж выполнит анимацию.

## ✨ Возможности

- Набор встроенных эмоций (танцы, жесты, реакции).
- **Кастомные эмоции** — можно добавлять свои анимации.
- Колесо быстрого доступа для мгновенного отыгрыша.

> Нужен **клиентский мод Emotecraft**. Игроки без него могут не видеть твою эмоцию — для полноценного взаимодействия мод желателен всем.

## 💡 Советы

1. Забей в колесо 6–8 любимых эмоций для быстрых реакций.
2. Эмоции — отличный способ «оживить» скриншоты и стримы.
3. Хорошо сочетается с [кастом-моделью](cpm.md) для полного образа.
