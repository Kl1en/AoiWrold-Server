# ⚙️ Grind Enchantments — работа с чарами на точиле

[← Каталог модов](README.md)

Расширяет **точильный камень (grindstone)**: теперь через него можно управлять зачарованиями, а не только снимать их «в никуда».

## 🔧 Что умеет

- **Извлечение чар** с предмета на **книгу** — снятое зачарование не пропадает, а сохраняется.
- **Перенос чар** между предметами/книгами.
- Гибкая работа с зачарованиями без дорогих операций на наковальне.

> Точное поведение (что и как переносится, стоимость) смотри в игре на точиле — интерфейс подскажет.

## 💡 Зачем

- Сохранить ценное зачарование с ломающегося предмета.
- Пересобрать «идеальный» инструмент, перекинув нужные чары.
- Экономия ресурсов по сравнению с наковальней.

Отлично сочетается с [NeoEnchant](neoenchant.md) — можно перетасовывать десятки новых зачарований.
