# 🏭 PolyFactory — механика и фабрики

[← Каталог модов](README.md) · [Обзор техники](../10-Техника.md)

Индустриальный мод в духе **Create**: строишь производственные цепочки на **вращении** и **электричестве**. Всё серверное — работает даже на ванильных клиентах.

## ⚙️ Источники энергии (вращение)

Механизмам нужна **сила вращения**. Источники по возрастанию:
| Источник | Особенность |
|----------|-------------|
| **Hand Crank** (рукоятка) | крутишь вручную — для старта |
| **Steam Engine** (паровой двигатель) | стабильная механическая мощь на топливе/воде |
| **Windmill** (ветряк) | «бесплатная» энергия от ветра |
| **Electric Motor** | превращает электричество во вращение |
| **Wither Skull Generator** | экзотический источник энергии |

## 🔩 Передача вращения

- **Axle** (вал), **Axle with Gear / Large Gear** — передают и меняют вращение.
- **Gearbox** (коробка передач) — раздаёт вращение по направлениям.
- **Clutch** (муфта) и **Gearshift** — управление и реверс.
- **Turntable** (поворотный стол).

Следи за **нагрузкой (stress)**: чем больше машин — тем больше нужно мощности. Контролируй **Stressometer** и **Tachometer**.

## 🛠️ Машины

| Машина | Что делает |
|--------|-----------|
| **Grinder** (мельница) | перемалывает (дробление ресурсов) |
| **Mechanical Press** | прессует/штампует |
| **Mechanical Miner** | автоматическая добыча |
| **Mechanical Placer** | ставит блоки |
| **Mechanical Mixer** | смешивает |
| **Mechanical Crafter** | **автокрафт** предметов |
| **Mechanical Planter** | сажает растения |

## 📦 Транспорт и хранение

- **Item Conveyor / Sticky Conveyor** — конвейерные ленты.
- **Funnel / Slot-Aware Funnel** — загрузка/выгрузка предметов.
- **Fan** (вентилятор), **Mechanical Ejector** (выталкиватель), **Item Splitter** (разделитель), **Item Packer** (упаковщик).
- **Wooden / Deep Storage Container**, **Item Output Buffer** — хранение и буферизация.

## 💡 Логика и индикация

- **Nixie Tube** (+ Controller/Connector) — цифровые табло.
- **Hologram Projector**, **Gauge**, **Digital Clock**.
- **Redstone Input/Output**, **Block Observer**, **Item Reader**, **Text Input** — связь с редстоуном и автоматика.
- **Speaker / Record Player** — звук.

## ⚡ Кабели

**Cable / Gated Cable** — передают энергию между устройствами (в т.ч. электромотор ↔ генератор).

## 🚀 Путь развития

1. **Рукоятка → Grinder** — первая обработка.
2. **Паровой двигатель / ветряк** — стабильная мощность.
3. **Электричество** (генератор + кабели + электромотор) — масштабирование.
4. **Mechanical Crafter + конвейеры** — полноценная автоматизированная линия.
