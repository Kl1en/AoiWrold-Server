# ✨ NeoEnchant — зачарования

[← Каталог модов](README.md)

Около **полусотни** новых зачарований сверх ванильных. Получаются через стол зачарования, книги и наковальню (как обычные чары). Ниже — по группам с описанием эффекта.

## ⛏️ Добыча и инструменты
| Зачарование | Эффект |
|-------------|--------|
| **Vein Miner** | ломает всю связанную жилу руды разом |
| **Timber** | валит дерево целиком одним ударом |
| **Auto Smelt** | добытое сразу переплавляется |
| **Mining+** | копает в области 3×3 |
| **Harvest** | сбор урожая по площади |
| **Bedrock Breaker** | ломает бедрок (150 прочности за блок) |
| **Spawner Touch** | позволяет забрать спавнер мобов |
| **Builder Arm** | увеличенная дальность установки/ломания |

## 🏃 Передвижение
| Зачарование | Эффект |
|-------------|--------|
| **Step Assist** | автозаход на блоки без прыжка |
| **Lava Walker** | ходьба по лаве |
| **Leaping / Ethereal Leap** | усиленный прыжок / рывок |
| **Fast Swim** | скорость в воде (Dolphin's Grace) |
| **Agility** | скорость передвижения |
| **Wind Propulsion / Velocity** | ускорение на элитрах |
| **Voidless** | кратковременная левитация при падении в бездну |

## ⚔️ Ближний бой
| Зачарование | Эффект |
|-------------|--------|
| **Life Steal** | крадёшь здоровье с врагов |
| **Poison Aspect** | отравляет цель |
| **Death Touch** | жертвует предмет ради бесконечного урона |
| **Fear** | накладывает Тьму/страх на врагов |
| **Reach** | увеличенная дальность атаки |
| **Attack Speed** | скорость атаки |
| **Critical Hit / Xp Boost** | шанс крита / больше опыта с убийств |

## 🏹 Лук и стрелы
| Зачарование | Эффект |
|-------------|--------|
| **Explosive Arrow** | взрыв при попадании |
| **Storm Arrows** | молния при попадании |
| **Echo Shot** | звуковой удар по площади |
| **Eternal Frost** | заморозка и замедление |
| **Accuracy Shot** | стрелы летят прямо без гравитации |
| **Rebound** | стрелы рикошетят от стен |

## 🛡️ Защита и утилити
| Зачарование | Эффект |
|-------------|--------|
| **Auto Feed** | автокормёжка каждые 3 минуты |
| **Bright Vision** | ночное зрение |
| **Armored / Kinetic Protection** | защита (в т.ч. от урона столкновений элитр) |
| **Life+** | доп. сердца |
| **Venom Protection** | защита от негативных эффектов |

## 💀 Проклятия (для челленджа/торговли)
**Curse of Fragility** (быстрее теряет прочность), **Curse of Enchant** (нельзя менять чары), **Curse of Clumsiness** и др.

> Это не полный список — в NeoEnchant есть и экзотика (Midas Touch — превращение блоков в золото, Gungnir Breath, Teluric Wave и пр.). Полный набор смотри в игре на столе зачарования.
