# 🦊 AoiPets — Лисы Аой

[← Каталог модов](README.md) · [Обзор питомцев](../03-Питомцы.md)

Фирменная система питомцев сервера, написанная специально для AoiWorld (в духе MCPets). Декоративный компаньон-лиса, который ходит за тобой.

## 🦊 Что это

Призванная лиса ведёт себя как прирученный волк: **следует за хозяином**, телепортируется, если отстала, и показывает **статус над головой**:
- «Следует за хозяином»
- «Сидит (ждёт)»
- «Атакует»

Модель кастомная, с анимацией, видна **всем игрокам** (в т.ч. на ванильном клиенте) — клиентские моды не нужны.

## 🎨 6 лис

| ID | Лиса |
|----|------|
| `red_fox` | Красная |
| `fall_fox` | Осенняя |
| `gray_fox` | Серая |
| `pink_fox` | Розовая |
| `purple_fox` | Фиолетовая |
| `arctic_fox` | Арктическая |

## ⌨️ Команды

| Команда | Действие |
|---------|----------|
| `/aoifox` | открыть меню выбора лисы |
| `/aoifox summon <лиса>` | призвать конкретную лису |
| `/aoifox dismiss` | отозвать лису |
| `/aoipet` | алиас `/aoifox` |

## 🔐 Доступ по рангам

Лисы выдаются через права (строгий режим — «всё по рангам»):

| Право | Что даёт |
|-------|----------|
| `aoipets.use` | доступ к меню и команде |
| `aoipets.summon.<лиса>` | конкретная лиса (напр. `aoipets.summon.arctic_fox`) |
| `aoipets.summon.*` | все лисы сразу |

В меню недоступные лисы показаны как заблокированные (🚫 «Нет доступа»). Как получить редкую лису (ранг/донат/награда) — смотри в анонсах сервера.

## 💡 Особенности

- Лиса **мирная и декоративная** — это не боевой пет (для боя есть [Battle Fox](battle-fox.md)).
- Одновременно активна одна лиса; призыв новой заменяет текущую.
- Лиса автоматически возвращается при заходе на сервер и убирается при выходе.
