# 🛋️ Мебель и декор

[← Каталог модов](README.md) · [Обзор строительства](../09-Строительство.md)

Два больших мебельных мода + фирменные плюшки. Всё серверное, видно на ванильных клиентах.

## 🪑 PolyDecorations

Огромный набор мебели (131 блок + предметы). Что внутри:
- **Полки (Shelves)** — под все виды дерева, для выкладки предметов.
- **Почтовые ящики (Mailbox)** — по дереву каждого типа.
- **Сиденья (Seat)** — можно сидеть.
- **Статуи (Statue)** — декоративные фигуры.
- **Холст (Canvas)** — рисование/оформление.
- **Узел для поводка (Leash Knot)** и прочий декор.

Дерево всех пород (дуб, берёза, ель, вишня, мангр, бамбук, багровое/искажённое и т.д.) — мебель под любой стиль.

## 🪚 TSA: Decorations & Furniture

Ещё больше мебели (258 блоков!) с собственной станцией крафта:
- **Carpentry Table** (столярный стол) — станция для создания мебели TSA.
- **Cut Planks** — «резаные» доски всех пород как база.
- **Скамьи (Iron Bench)**, столы, стулья и десятки предметов интерьера по всем породам дерева.

## 🧸 NM Plushies — плюшки

10 коллекционных плюшек (перенесены со старого сервера), ставятся на пол с плавным поворотом:
`Alivepond · Alloy Boy · Awesome Guy · Bug Boy · Colonel Arkansas · Honey Badger · Night Bat · Odin · Pulse · The Hunk`

## 🖼️ Оформление стен
- **Image2Map** — картинка → карта на стену (арты, портреты, баннеры).
- **Craftable Invisible Item Frames** — невидимые рамки для чистой выкладки.
- **Armor Poser** — точные позы стоек с бронёй для музеев/выставок.

## 💡 Советы

1. Комбинируй **PolyDecorations + TSA** — вместе закрывают почти любой интерьер.
2. TSA-мебель делается на **Carpentry Table** — поставь её первой.
3. Полки + невидимые рамки + Image2Map = топ для «музеев» и уютных баз.
4. Плюшки и статуи оживляют жилые комнаты.

> Крафты смотри в игре через [Polydex](../13-Качество-жизни.md).
