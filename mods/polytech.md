# 🔌 PolyTech — энергия и переработка

[← Каталог модов](README.md) · [Обзор техники](../10-Техника.md)

Классический тех-мод про **электроэнергию** и **обогащение руды**. Есть встроенный гайд — **PolyTech Handbook** (скрафти и читай прямо в игре).

## ⚡ Энергосистема

| Блок | Роль |
|------|------|
| **Generator** | базовая генерация энергии (на топливе) |
| **Solar Panel** | «бесплатная» энергия от солнца (днём) |
| **Battery Box** | аккумулятор — хранит энергию про запас |
| **Energy Cable** | провода для передачи энергии |
| **Wireless Transmitter / Receiver** | беспроводная передача энергии |

Схема простая: **источник → кабель/аккумулятор → машины-потребители**.

## ⛏️ Переработка руды (главная фишка)

Обогащение **удваивает** выход металла:

```
Руда → [Macerator] → 2× Пыль → [переплавка] → 2× слитка
Слиток → [Compressor] → Пластина
```

| Машина | Что делает |
|--------|-----------|
| **Macerator** (дробилка) | руда → **пыль** (×2 выход) |
| **Compressor** (пресс) | слиток → **пластина** |
| **Electric Furnace** | быстрая переплавка на энергии |

Материалы: **Iron / Gold / Copper** → Dust → Plate. Пластины и пыль идут в продвинутые крафты.

## 🤖 Автоматизация

- **Digital Crafter + Pattern Terminal** — автокрафт по сохранённым шаблонам.
- **Quarry** (карьер) и **Chunk Destroyer** — автоматически выкапывают целые области. Ставится **Quarry Frame** (рамка), внутри неё идёт добыча.
- **Villager Grinder / Combiner** — механики работы с жителями (Villager Soul).
- **Inverted Redstone Lamp**, беспроводной редстоун — для схем.

## 🚀 Путь развития

1. Поставь **генератор/солнечку + Battery Box**, разведи **кабели**.
2. Добавь **Macerator** и **Compressor** — обогащай руду ×2.
3. Поставь **Electric Furnace** для быстрой плавки.
4. Автоматизируй добычу **Quarry**, крафт — **Digital Crafter**.
5. Читай **PolyTech Handbook** — он ведёт по прогрессии.

> Отлично сочетается с [PolyFactory](polyfactory.md) и [Digital Chest](digital-chest.md) как складом.
