# 🤗 Headpats — погладь по голове

[← Каталог модов](README.md) · [Обзор социалки](../12-Социалка.md)

Милая социальная фича: **гладь других игроков по голове** 🥺. Чистая ламповость для комьюнити.

## 🔧 Как работает

1. Наведись на игрока.
2. Используй действие «погладить» (взаимодействие).
3. Над игроком проигрывается приятная анимация/эффект.

## 💡 Зачем

- Тёплое взаимодействие между игроками.
- Отлично для дружеской атмосферы, стримов и скриншотов.
- Идеально ложится в уютный настрой сервера Aoi 🦊.
