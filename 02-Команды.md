# 📋 Все команды

[← На главную](README.md)

Шпаргалка по основным командам. Не помнишь точный синтаксис? Введи `/` и жми **Tab**.

## 🔐 Вход
| Команда | Действие |
|---------|----------|
| `/register <pass> <pass>` | регистрация (первый вход) |
| `/login <pass>` | вход |
| `/changepassword <old> <new>` | смена пароля |

## 🏠 Дом, телепорты, варпы
| Команда | Действие |
|---------|----------|
| `/sethome [имя]` | поставить дом |
| `/home [имя]` | телепорт домой |
| `/homes` | список твоих домов |
| `/delhome <имя>` | удалить дом |
| `/spawn` | на спавн |
| `/warp <имя>` | телепорт на варп (напр. `/warp spawn`) |
| `/warps` | список варпов |
| `/tpa <игрок>` | запрос телепорта к игроку |
| `/tpaccept` · `/tpdeny` | принять / отклонить запрос |
| `/back` | вернуться на прошлую точку (после смерти/тп) |
| `/rtp` | случайная телепортация (дикий мир) |
| `/nick <ник>` | сменить отображаемый ник |

> Количество домов зависит от твоего ранга. Подробнее — [Дом и варпы](06-Дом-и-варпы.md).

## 🦊 Питомцы — Лисы Аой
| Команда | Действие |
|---------|----------|
| `/aoifox` | открыть меню лис |
| `/aoifox summon <лиса>` | призвать лису |
| `/aoifox dismiss` | отозвать лису |
| `/aoipet` | алиас `/aoifox` |

Лисы: `red_fox`, `fall_fox`, `gray_fox`, `pink_fox`, `purple_fox`, `arctic_fox`. Доступ к ним зависит от прав/ранга — см. [Питомцы](03-Питомцы.md).

## ⭐ Навыки, экономика, магазины
Эти системы в основном работают **через меню (GUI)**. Открывай командой и жми Tab для подсказок:
| Система | Как открыть |
|---------|-------------|
| Навыки PeakSkills | меню навыков (коллекции, статы, петы) — [подробнее](04-Навыки.md) |
| Кошелёк / баланс | посмотреть деньги и историю — [Экономика](05-Экономика.md) |
| Магазины | купля-продажа предметов |
| Digital Chest | безлимитное хранилище с поиском |
| Backpack | личный рюкзак |

## 🎙️ Голос и эмоции
| Действие | Как |
|---------|-----|
| Голосовой чат | работает автоматически (нужен мод Simple Voice Chat на клиенте) |
| Эмоции | клавиша меню эмоций (Emotecraft), по умолчанию **B** |
| Погладить игрока | навести на игрока и использовать (Headpats) |

## 🛠️ Полезное
| Команда/действие | Действие |
|---------|----------|
| `Tab` (в списке игроков) | посмотреть онлайн и ранги |
| ПКМ по грядке | быстрый сбор урожая (Harvest with ease) |
| ПКМ по двери/калитке | быстро открыть (Openable) |
| ПКМ по вейстоуну | сеть телепортов (Waystones) |
| Shift+ПКМ по мобу | подобрать моба (если разрешено) |
