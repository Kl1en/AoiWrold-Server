# 🌸 Вики сервера AoiWorld

Добро пожаловать в официальную вики сервера **AoiWorld** — уютного мира сообщества витубера **Aoi Hinode** 🦊

> Сервер построен на ядре **Fabric 1.21.11** и работает так, что зайти можно и с обычного (ванильного) клиента — весь кастомный контент подгружается через ресурспак сервера. Ставить моды НЕ нужно.

## ℹ️ Коротко о сервере

- **Версия:** Minecraft 1.21.11 (Fabric)
- **Режим:** выживание (hard), с большим количеством контентных модов
- **Вход:** по вайтлисту, авторизация по паролю (`/register`, `/login`)
- **Клиент:** подойдёт любой — ванильный, Fabric, лаунчер на выбор. Ресурспак прилетает автоматически.
- **Онлайн:** до 80 игроков

*Эта вики описывает то, что реально установлено на сервере (132 модуля). Точные команды всегда можно подсмотреть в игре — начни вводить `/` и жми **Tab**.*
